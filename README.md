## Getting Started   

1. **Clone the repository**   
   ```bash

   cd python
   ```

2. **Dependencies** (recommended: Conda or venv)   
   ```bash
   conda create -n py_tut python=3.10
   conda activate py_tut
   pip install python sqlalchemy
   ```
